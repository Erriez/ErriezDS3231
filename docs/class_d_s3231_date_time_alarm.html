virtual

Set and enable Alarm 1.

Alarm 1 contains several alarm modes which can be configured with the alarmType parameter. Unused matches can be set to zero. The alarm interrupt must be enabled after setting the alarm, followed by clearing the alarm interrupt flag.

Parameters
alarmTypeAlarm 1 types:
Alarm1EverySecond
Alarm1MatchSeconds
Alarm1MatchMinutes
Alarm1MatchHours
Alarm1MatchDay
Alarm1MatchDate
dayDateAlarm match day of the week or day of the month. This depends on alarmType.
hoursAlarm match hours.
minutesAlarm match minutes.
secondsAlarm match seconds.
Return values
Success
FailureI2C transfer failed.
